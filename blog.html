Important FAQ

What is the Document Object Model(DOM)?

DOM is a programming interface for interacting with HTML documents which represents itself as a tree structure. Where each node represents as an element.

How do you select an element from the DOM?

We can select an element from the DOM by getElementById, getElementsByTagName, querySelector, querySelectorAll, getElementsByClassName etc.

What is event delegation in the context of the DOM, and why it is useful?

It refers to the system of attaching an event listener to a parent element instead of each child element. By using this, we can write more efficient code, as we only need to manage a single event listener on the parent element.

How do you manipulate an element`s attributes and styles using the DOM?

For manupulating an element's attributes we use getAttribute, removeAttribute and setAttribute. For manupulating an element's style we use element.classList.add, element.classList.remove and element.style.propertyName.
